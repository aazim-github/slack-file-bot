# slack-file-bot

-- Go-slack-file-bot -- File Upload Bot integrated with Slack using Golang 

-- Description -->

- This project is build using Golang and 'github.com/slack-go/slack' API exhibiting a file upload service which can be used for personal file uploader service in slack platform. 
- Using the 'github.com/slack-go/slack' API which is a Slack API for Golang and the logic for the API to upload the specified file on to the Slack personal group is taken care with Golang.
